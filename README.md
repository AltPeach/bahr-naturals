# Bahr Naturals

A fully responsive, accessible website for Bahr Naturals - handcrafted natural soaps honoring South Sudanese heritage.

## 🌊 Project Overview

Bahr Naturals is a culturally authentic e-commerce website that bridges South Sudanese heritage with Canadian craftsmanship. The site features clean minimalist design, cultural storytelling, and full e-commerce functionality.
